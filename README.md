# Uniportal

University portal backend built with Spring Boot.

## Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Liquibase
  - Cloudflare R2 (S3)

## Features

- Admin management
- Student management
- Teacher management
- Groups and departments
- Subjects and lessons
- Image storage via Cloudflare R2
